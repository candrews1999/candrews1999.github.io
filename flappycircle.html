Flappy Circle

This HTML game is based on the popular app Flappy Bird. The animation and game mechanics are programmed using the D3.js library and Javascript. The goal is to help the yellow circle avoid the green pipes by pressing the spacebar to raise it to the right height. I also programmed a scoreboard that records high scores for each session of play.

Notes from the Programming Process

  • D3 data joins are used for accessing the elements from the array list of pipes that are still in still on screen.
  • Buttons were created and styled with HTML and CSS but their effects were implemented in Javascript and D3.
  • The game is programmed to be over when the circle falls below the bottom of the screen or when the circle overlaps in some capacity with a pipe.
  • The animation and spawning of pipes are timed using the Javascript setInterval method.
  • Pipes travel leftward with every tick from setInterval and are removed once offscreen to prevent unnecessary data tracking.
  • Insertion sort was used to update the leaderboard with each incoming high score.
